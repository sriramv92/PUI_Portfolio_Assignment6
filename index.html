BEAT BOX APP Design Concept

UI DESIGN | INTERACTION DESIGN

User Interface design concepts for the BEAT BOX mobile application.


Beat Box is a mobile application concept. The aim of this application is to connect to the CADDHM designed as a technial project. The app interfaces are designed to be minimalistic and easy to use. The UI designs have functionalities that allow one to record, hear, playback and delete the heart beat recordings taken from the device.

Tools Used

  • Adobe Illustrator
  • Adobe Photoshop
  • Invision - Prototyping
